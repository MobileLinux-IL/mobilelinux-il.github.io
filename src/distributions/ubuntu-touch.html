אפליקציות Waydroid

אפליקציות ושירותי אנדרואיד שרצים דרך Waydroid. ללא שירותי Play כברירת מחדל.

Waydroid אינו תומך בהעברת מיקום, כך שכל דבר שמבוסס מיקום לא יעבוד.

אם צריך תמיכת NFC ב-Waydroid יש להריץ את הפקודה "waydroid prop set persist.waydroid.uevent true" ולהפעיל מחדש את Waydroid במידת הצורך.

אפליקציה / שירותמצבהערות
בנק דיסקונטחלקיהאפליקציה עובדת ב-Waydroid אך ללא כניסה בטביעת אצבע.
רב-קו אונלייןעובדאפליקציית האנדרואיד עובדת ב-Waydroid (דורשת "waydroid prop set persist.waydroid.uevent true"). אני יבנה אפליקציית טעינת רב-קו מקורית, כבר ביצעתי הנדסה לאחור של מה שצריך — ראו: openkav
